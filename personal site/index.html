Projects

Automated AWS CloudWatch Reporting System

This alarm system is implemented in AWS to generate a daily report of my CloudWatch alarm activities and CloudWatch agents in my EC2 instances. I setup EventBridge to send a log to a CloudWatch log group everytime my alarms are triggered(changed states to ALARM). I then created a lambda function that queries all the log events in the past 24hrs and writes them to an excel csv file. The lambda function will also get the metrics collected by each active CloudWatch agents that are active in all my EC2 instances. These csv files are stored in an S3 bucket. Finally, these csv files will be sent via email using SES. I scheduled an EventBridge trigger the lambda function every 24hrs to send me these daily reports.

Rogue Roullete

Rogue roullete is a game that I developed as a team of 6 people. Over the course of 3 months, we routinely release new versions of our game starting from alpha to gold and documented user feedback and changes that were made in a game design document. Stock Market Prediction

I modeled a recurring neural network that predicts the future stock prices using the keras module. I also designed the optimal data preprocessing technique by scaling the data using a MinMaxScaler and reduce noise by implementing a moving average algorithm. Finally, I tested and tuned hyperparameter on both the testing and validation dataset.

BitTorrent Client

Co-developed a BitTorrent client in Go-Lang that follows the BitTorrent protocol. Features include reading the .torrent metainfo file, communicating with the tracker, optimistic unchoking, seeding a specific file, downloading files, and interacting with existing BitTorrent clients.
Client has not been used for malicious purposes. This project is a showcase of my understanding of network protocols
